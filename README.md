# CLO Management System

A comprehensive Course Learning Outcome (CLO) Management System for educational institutions to track, assess, and analyze student learning outcomes aligned with Program Learning Outcomes (PLOs).

## Features

- **Role-Based Access Control**: Admin, Faculty, and Student roles
- **Program & Course Management**: Define programs, PLOs, courses, and CLOs
- **Assessment Tools**: Create quizzes, assignments, midterms, finals, projects, and labs
- **CLO-PLO Mapping**: Map course outcomes to program outcomes with correlation strength
- **Automated Attainment Calculation**: Calculate student-level and course-level CLO attainment
- **Reports & Analytics**: Export Excel reports, visualize attainment matrices
- **Student Progress Tracking**: Students can view their CLO attainment and scores

## Technology Stack

- **Backend**: Flask (Python 3.10)
- **Database**: MySQL
- **Frontend**: Bootstrap 5, HTML, CSS, JavaScript
- **Authentication**: Flask-Login (session-based)
